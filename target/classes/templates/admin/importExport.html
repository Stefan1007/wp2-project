Import Quizzes

Import quizzes from a JSON file.

Import Format

The JSON file should be an array of quiz objects with the following structure:

[
  {
    "title": "Quiz Title",
    "duration": 15,
    "questions": [
      {
        "text": "Question text?",
        "optionA": "First option",
        "optionB": "Second option",
        "optionC": "Third option",
        "correctAnswer": "A"
      },
      // More questions...
    ]
  },
  // More quizzes...
]

Tips

  • Export quizzes for backup purposes
  • Share exported quizzes with other instructors
  • Import quizzes from external sources
  • Make sure imported quizzes follow the correct JSON format
